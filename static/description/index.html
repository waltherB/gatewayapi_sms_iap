GatewayAPI SMS Connector for Odoo 17

Send SMS via GatewayAPI.eu using IAP Alternative Provider

Overview

Integrate your Odoo 17 instance with GatewayAPI.eu, a reliable and feature-rich SMS gateway. This module leverages the robust IAP Alternative Provider framework from the Odoo Community Association (OCA), ensuring a stable and extensible solution for your SMS needs.

Send SMS notifications, marketing messages, and alerts directly from Odoo, with support for emojis, automated balance checks, and low credit warnings.

Key Features

Easy Configuration

Securely set up your GatewayAPI credentials within Odoo's familiar interface. Default URL pre-configured to https://gatewayapi.eu.

Emoji Support (UCS-2)

Messages containing emojis or special characters are automatically detected and sent using UCS-2 encoding.

Secure Token Handling

API Token field includes a show/hide toggle for enhanced security during configuration.

Balance Check & Alerts

Test your connection, view current GatewayAPI credits, and set up automated checks with low credit notifications via Odoo's messaging system.

IAP Alternative Provider

Built upon the flexible IAP Alternative Provider (OCA) module for future-proof integration.

Multi-Language

Includes initial translations for English and Danish. Easily extendable for other languages.

Prerequisites

This module depends on the iap_alternative_provider module from the Odoo Community Association (OCA). You must install this module first. You can find it in the OCA `server-tools` repository: https://github.com/OCA/server-tools/tree/17.0/iap_alternative_provider

Ensure this OCA module is added to your Odoo addons path and installed before installing the GatewayAPI SMS Connector.

Installation & Configuration

  1. Ensure the iap_alternative_provider (OCA) module is installed.
  2. Install the "GatewayAPI SMS Connector (IAP Alternative)" module.
  3. Navigate to Settings » Technical » Alternative Providers.

    (Screenshot: Navigating to Alternative Providers menu - Replace with your actual screenshot)

  4. Click "Create" to add a new alternative provider configuration:
    • Name: E.g., "GatewayAPI Main Account".
    • Provider: Select "GatewayAPI" from the dropdown.
    • GatewayAPI Account Label: A friendly name for this configuration.
    • GatewayAPI Base URL: Defaults to https://gatewayapi.eu. Change if needed.
    • Default Sender Name: Your desired SMS sender ID (alphanumeric or numeric, as per GatewayAPI rules).
    • API Token: Your API token from your GatewayAPI.eu dashboard. Use the eye icon to show/hide.
    • Available for services: Select "sms".

    (Screenshot: Configuring the GatewayAPI Provider - Replace with your actual screenshot)

  5. Click "Check Credentials" to test your API token and view your current GatewayAPI balance.
  6. (Optional) Configure "Automated Balance Check & Notifications":
    • Enable balance checks.
    • Set a minimum credit limit for alerts.
    • Define the check interval.
    • Select a notification channel or users to receive alerts.
  7. Save the Alternative Provider record.
  8. Navigate to Settings » Technical » IAP Accounts.
  9. Create a new IAP Account (or edit an existing one for SMS):
    • Service Name: Select or type "sms".
    • Alternative Provider: Select the GatewayAPI provider record you just configured.

    (Screenshot: Linking IAP Account to Provider - Replace with your actual screenshot)

  10. Save the IAP Account.

Your Odoo instance is now configured to send SMS messages via GatewayAPI!

Usage

Once configured, Odoo will use GatewayAPI for sending SMS messages through its standard SMS framework. This includes:

  • Sending SMS from Contact forms.
  • Sending SMS via CRM lead/opportunity automation.
  • Using the SMS composer in various parts of Odoo.
  • Programmatic SMS sending from custom modules (using self.env['sms.api']._send_sms(...)).

The module automatically handles UCS-2 encoding for messages containing emojis or special characters.
